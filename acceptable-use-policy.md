# Правила использования Cynic VPS

Редакция от: 01.03.2026

Настоящие Правила определяют допустимые и запрещённые способы использования сервиса Cynic VPS.

Используя сервис, профиль подключения, бота, сайт, платёжную форму или поддержку Cynic VPS, пользователь соглашается соблюдать настоящие Правила.

## 1. Общие положения

1.1. Cynic VPS предоставляет цифровой доступ к персональному профилю подключения на срок и условиях выбранного тарифа.

1.2. Пользователь обязан использовать сервис добросовестно, законно и в пределах выбранного тарифа.

1.3. Пользователь самостоятельно несёт ответственность за свои действия при использовании сервиса.

## 2. Лимиты тарифа

2.1. Количество устройств, срок действия доступа и иные параметры определяются выбранным тарифом.

2.2. Пользователь не вправе обходить технические ограничения тарифа.

2.3. При превышении лимитов сервис вправе ограничить, приостановить или прекратить доступ.

## 3. Передача доступа третьим лицам

Пользователю запрещается:

- передавать профиль подключения третьим лицам;
- публиковать ссылку профиля в открытом доступе;
- продавать, сдавать в аренду или иным образом распространять доступ;
- использовать профиль совместно с лицами, не входящими в условия выбранного тарифа;
- создавать копии профиля для обхода лимитов устройств.

## 4. Запрещённые действия

Пользователю запрещается использовать сервис:

- для противоправных действий;
- для атак на сайты, серверы, приложения или информационные системы;
- для спама, фишинга, мошенничества или массовых рассылок;
- для распространения вредоносного программного обеспечения;
- для нарушения прав третьих лиц;
- для обхода ограничений платёжных систем, банков, маркетплейсов или иных сервисов;
- для автоматизированной регистрации аккаунтов, накруток, парсинга или иных злоупотреблений;
- для действий, нарушающих законодательство Российской Федерации.

## 5. Запрещённый трафик

Сервис вправе ограничивать или блокировать:

- torrent-трафик;
- подозрительную массовую активность;
- вредоносные соединения;
- действия, создающие повышенную нагрузку на инфраструктуру;
- подключения, похожие на злоупотребление сервисом.

## 6. Ограничение доступа

Сервис вправе ограничить, приостановить или прекратить доступ пользователя без возврата денежных средств, если пользователь:

- нарушил настоящие Правила;
- передал профиль третьим лицам;
- превысил лимит устройств;
- пытался перепродать доступ;
- использовал сервис для запрещённых действий;
- создавал угрозу стабильности, безопасности или репутации сервиса.

## 7. Проверка нарушений

7.1. Сервис может использовать технические данные для выявления превышения лимитов, передачи профиля третьим лицам, злоупотреблений и нарушений настоящих Правил.

7.2. Сервис не предназначен для хранения истории посещённых пользователем сайтов.

7.3. Проверка нарушений проводится только в объёме, необходимом для защиты сервиса, пользователей и инфраструктуры.

## 8. Контакты
