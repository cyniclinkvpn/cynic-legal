# Политика конфиденциальности Cynic VPS

Редакция от: 01.03.2026

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании сервиса Cynic VPS, для каких целей они используются и как пользователь может обратиться по вопросам обработки данных.

Используя сервис, бот, сайт, платёжную форму, профиль подключения или поддержку Cynic VPS, пользователь подтверждает, что ознакомился с настоящей Политикой.

## 1. Общие положения

1.1. Cynic VPS обрабатывает только те данные, которые необходимы для работы сервиса, активации подписки, обработки платежей, поддержки пользователей, предотвращения злоупотреблений и соблюдения требований законодательства.

1.2. Сервис не предназначен для сбора избыточных данных о пользователе.

1.3. Сервис не запрашивает паспортные данные, адрес проживания, сведения о здоровье, политических взглядах, религии, биометрические данные или иные специальные категории персональных данных.

## 2. Какие данные могут обрабатываться

Сервис может обрабатывать следующие данные:

- Telegram ID пользователя;
- Telegram username, если он доступен;
- имя профиля Telegram, если оно отображается в интерфейсе;
- сведения о выбранном тарифе;
- статус подписки;
- дата начала и окончания подписки;
- количество подключённых устройств;
- технические идентификаторы подписки и профиля;
- сведения об оплате, включая дату, сумму, статус платежа и идентификатор платежа;
- обращения пользователя в поддержку;
- технические события, связанные с активацией, обновлением, продлением, удалением устройств и восстановлением доступа.

## 3. Технические данные

3.1. Для обеспечения работы сервиса могут обрабатываться технические данные, необходимые для активации и поддержки профиля подключения.

3.2. Сервис не предназначен для хранения истории посещённых пользователем сайтов.

3.3. Сервис не использует данные пользователя для продажи третьим лицам.

3.4. Сервис может использовать технические данные для предотвращения злоупотреблений, превышения лимитов тарифа, передачи профиля третьим лицам, атак, спама и иных нарушений правил использования.

## 4. Цели обработки данных

Данные обрабатываются для следующих целей:

- регистрация пользователя в сервисе;
- создание и активация подписки;
- выдача профиля подключения;
- обработка платежей;
- продление доступа;
- контроль лимитов устройств;
- техническая поддержка;
- восстановление доступа;
- рассмотрение обращений о возврате;
- предотвращение злоупотреблений;
- улучшение стабильности и безопасности сервиса;
- соблюдение требований законодательства.

## 5. Платежи

5.1. Оплата может обрабатываться через платёжные формы, банки, платёжные системы или иных платёжных посредников.

5.2. Сервис может получать техническую информацию о платеже: сумму, дату, статус, идентификатор платежа и сведения, необходимые для активации подписки.

5.3. Сервис не хранит полные данные банковских карт пользователя.

## 6. Поддержка

6.1. При обращении в поддержку пользователь может добровольно предоставить сведения, необходимые для решения проблемы.

6.2. Поддержка может запросить Telegram ID, username, дату оплаты, тариф, описание проблемы, сведения об устройстве, приложении и скриншоты, если это необходимо для проверки обращения.

6.3. Пользователь не должен отправлять в поддержку лишние персональные данные, не относящиеся к обращению.

## 7. Передача данных третьим лицам

Данные могут передаваться третьим лицам только в случаях, необходимых для работы сервиса, включая:

- платёжные системы и платёжных посредников;
- хостинг-провайдеров и технических подрядчиков;
- сервисы аналитики и мониторинга, если они используются;
- государственные органы, если такая передача требуется по закону.

## 8. Срок хранения данных

8.1. Данные хранятся столько, сколько необходимо для целей обработки, работы подписки, поддержки пользователя, разрешения спорных ситуаций, предотвращения злоупотреблений и соблюдения требований законодательства.

8.2. После прекращения использования сервиса данные могут быть удалены или обезличены, если их дальнейшее хранение не требуется для законных целей.

## 9. Права пользователя

Пользователь вправе:

- запросить информацию об обработке своих данных;
- попросить уточнить данные;
- попросить удалить данные, если их хранение больше не требуется;
- отозвать согласие на обработку данных;
- обратиться по вопросам конфиденциальности в поддержку.

## 10. Отзыв согласия и удаление данных

10.1. Пользователь может обратиться с запросом на отзыв согласия или удаление данных через поддержку: @cynic_support.

10.2. Отзыв согласия может привести к невозможности дальнейшего использования сервиса, если обработка данных необходима для работы подписки, оплаты, поддержки или соблюдения требований законодательства.

## 11. Безопасность данных

Сервис принимает разумные технические и организационные меры для защиты данных от несанкционированного доступа, изменения, раскрытия или уничтожения.

## 12. Изменение Политики

Сервис вправе изменять настоящую Политику. Новая редакция вступает в силу с момента размещения в боте, на сайте, GitHub-странице или ином официальном интерфейсе сервиса.

## 13. Контакты

По вопросам обработки данных пользователь может обратиться в поддержку: @cynic_support.
